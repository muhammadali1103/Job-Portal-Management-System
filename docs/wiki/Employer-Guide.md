# Employer Guide

## Employer Flow

1. Register / login
2. Open dashboard
3. Create job post
4. Set apply method (WhatsApp / Email / Website)
5. Save and manage jobs

## Job Form Highlights

- Role, category, location
- Salary and experience
- Company details
- Primary contact + country code
- WhatsApp country code + number (all countries supported)

## Manage Jobs

From user/employer job pages:
- Edit existing posts
- Delete posts
- Update apply method anytime
