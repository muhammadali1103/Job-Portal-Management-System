# Admin Guide

## Access

- Login URL: `/login`
- Admin credentials:
  - `admin@jobportal.com`
  - `11223344`

## Admin Capabilities

- Approve/reject/manage jobs
- Add/edit/delete categories
- Add/edit/delete locations (country + location)
- Manage users
- Manage job roles
- View analytics

## Locations

From `Admin -> Locations`:
- Add `Country`
- Add `Location` (city/area)

New locations appear in home page location filter.

## Job Posting

Admin can post jobs directly from `Admin -> Jobs -> Create`.

Apply methods supported:
- WhatsApp
- Email
- Website URL
